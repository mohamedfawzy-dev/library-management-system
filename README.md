# 📚 Library Management System | نظام إدارة مكتبة

A simple console-based library management system built using C++.
نظام بسيط لإدارة مكتبة باستخدام لغة ++C من خلال الـ Console.

---

## 💡 About the Project | فكرة المشروع

This system allows managing a small library by handling books and members.
السيستم بيساعدك تدير مكتبة صغيرة من خلال التحكم في الكتب والأعضاء.

---

## ⚙️ Features | المميزات

### 🔹 Books Management | إدارة الكتب

* Add new books

* View available books

* Track borrowed books

* إضافة كتب جديدة

* عرض الكتب المتاحة

* متابعة حالة الكتب

---

### 🔹 Members System | نظام الأعضاء

* Register new members

* Login system

* Change password

* تسجيل أعضاء جدد

* نظام تسجيل دخول

* تغيير كلمة المرور

---

### 🔹 Borrowing System | نظام الاستعارة

* Borrow books with limits

* Prevent duplicate borrowing

* Return books

* استعارة الكتب بحد أقصى

* منع استعارة نفس الكتاب مرتين

* إرجاع الكتب

---

### 🔹 Fine System | نظام الغرامات

* Add fines on return

* View fines

* Pay fines

* إضافة غرامات عند الإرجاع

* عرض الغرامات

* دفع الغرامات

---

## ▶️ How to Run | طريقة التشغيل

1. Compile the code using any C++ compiler

2. Run the program

3. Follow the menu instructions

4. اعمل Compile للكود باستخدام أي Compiler C++

5. شغّل البرنامج

6. اتبع التعليمات من القائمة
